# Crazy Dice

Dice game with combos, ante, shop, and boss rounds.

## Play on your phone (GitHub Pages)

1. **Push the repo to GitHub** (create a repo named e.g. `crazydice`, then push).
2. **Turn on GitHub Pages**:  
   Repo → **Settings** → **Pages** → under **Build and deployment**, **Source** = **GitHub Actions**.
3. **Deploy**: Push to `main` (or `master`). The “Deploy to GitHub Pages” workflow will build and publish.
4. **Open on your phone**:  
   `https://<YOUR_GITHUB_USERNAME>.github.io/crazydice/`  
   (replace `crazydice` if your repo has a different name).

Add the URL to your home screen for an app-like shortcut.

## Run locally

```bash
npm install
npm run dev
```

Build: `npm run build` (output in `dist/`).
